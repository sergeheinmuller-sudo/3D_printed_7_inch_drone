# 3D_printed_7_inch_drone
An open-source, 3D-printable FPV quadcopter frame for 7" or 8" props. Designed for PLA or PETG (PETG recommended for durability/heat resistance), assembled with M3 screws and nuts. Includes STL/CAD files, bill of materials, and build/tuning notes for makers who want a fully custom, printable alternative to carbon fiber frames.
